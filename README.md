# Action Bot API

[Action bot for Telegram](https://github.com/Surtt/action-telegram-bot/tree/main/action-bot)

### Stack:
- TypeScript
- Node.js
- Express.js
- Prisma
- Jest
- Eslint
- Prettier

### Environment variable for the database
DATABASE_URL=

### Run:
```npm ci && npm start```

### Lint
```npm run lint```

### Run the e2e tests
```npm run test:e2e```
